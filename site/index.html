Gravity reveal elements when entering the viewport
with CSS animations.

Get now Examples

gravity is a library to animate DOM elements as soon as they enter the window and are visible to the user. This library is similar and is inspired by Wow.js

Easy to use, fast and configurable

Instalation

Install now in your project, through npm or yarn, or download the source code in a ".zip"

#npm npm install gravity-library
#yarn yarn add gravity-library
#cdnJS https://cdn.jsdelivr.net/npm...
#cdnCSS https://cdn.jsdelivr.net/npm...
Download

Is made up of a base library (gravity.js) and a css animation library (gravity.css) although you can use any other animation library if you wish.

gravity.min.js 2,9Kb + gravity.min.css 3,3Kb = 6,2Kb!!
